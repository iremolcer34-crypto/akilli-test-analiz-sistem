# Akıllı Test Analiz Sistemi

Java ve SQLite kullanılarak geliştirilen Android tabanlı test analiz ve çalışma takip uygulamasıdır.

## Özellikler

- Test ve deneme sonuçlarının kaydedilmesi
- Kitap ve soru takibi
- Ders bazlı performans analizi
- Güçlü ve zayıf ders tespiti
- Yanlış soru kutusu yönetimi
- Net trendi ve istatistiksel raporlar
- Kişiselleştirilmiş çalışma önerileri
- TYT ve AYT analizlerinin ayrı değerlendirilmesi

## Kullanılan Teknolojiler

- Java
- Android Studio
- SQLite
- RecyclerView
- Material Design

## Amaç

Öğrencilerin çalışma süreçlerini takip etmelerini, eksik konularını belirlemelerini ve performanslarını analiz ederek daha verimli bir çalışma planı oluşturmalarını sağlamak.
